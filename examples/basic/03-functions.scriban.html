Function calls

There are built-in functions for common tasks.

My city is {{ address.city | uppercase }}.

My address has {{ address.lines | count }} lines.

My full address is {{ address.lines | join("\n") }}.

The developer can add custom functions.

My full address is {{ address.lines | distinct | prefix("- ") | join(sep="\n") }}.

Can also use traditional function call syntax

My full address is {{ join(prefix(distinct(address.lines), "- "), sep="\n") }}.

Functions can be qualified for disambiguation

My full address is {{ address.lines | collection.distinct | string.prefix("- ") | string.join(sep="\n") }}.
